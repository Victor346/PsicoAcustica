AUDIO BINAURAL

Psicoacústica y neurociencias de la audición

1

El sonido binaural es una forma de grabar sonido, la cual logra que el oyente perciba sonidos dentro de un espacio tridimensional, asemejándose a cómo escuchamos los seres humanos.

Llamado también sonido 8D pero esta es una expresión inventada para marketing, el nombre correcto es sonido 3D.

Es necesario escuchar con audífonos para poder percibir el audio 3d, el cual nuestro cerebro interpreta como si estuviéramos en un espacio real.

¿Cómo es que funciona?

Para lograr una grabación binaural, se captura el sonido de la forma más parecida posible a como nosotros lo hacemos. Para lograr esto, se necesita tener al menos dos micrófonos omnidireccionales dentro de cavidades similares a las que tienen nuestras orejas, manteniendo la misma distancia que hay entre ellas.

¿igual al estéreo?

Llamado también sonido 8D pero esta es una expresión inventada para marketing, el nombre correcto es sonido 3D.

El estéreo es un sonido que suena por dos canales donde la posición espacial se limita a izquierda y derecha. Para percibirlo no es necesario usar audífonos como en el sonido binaural.

Música Binaural

Es una herramienta muy útil para complementar contenido visual(como los videos 360º).

Con el sonido se enriquece la experiencia de inmersión. también tiene una aplicación muy grande en el campo de los videojuegos.

No es muy común su uso, debido a que no todos escuchan música con audífonos. En la música, se busca que esta, se logre escuchar igual en diferentes dispositivos que tal vez no puedan hacer uso de este recurso
